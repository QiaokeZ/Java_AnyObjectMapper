##### 前言
项目中有很多JavaBean类，少则5-10个，多则10-20个，其中大部分的字段都重复了，并且有些字段名称一样但类型不一样，导致不能重用，为了解决这些问题，写了个智能JSON解析，一个JavaBean类替代所有的类。
